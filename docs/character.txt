【キャラクターシステム】
・名前をもつ(キャラクター毎に固定)(2〜5文字)
・各ステータスを持つ
  Lv(1〜99)、HP(0〜999)、MP(0〜999)、SP(0〜10)、ATK(0〜999)、DEF(0〜999)、AGI(0〜999)、LUK(0〜999)
・装備をもつ(キャラクター毎に固定)
・操作キャラクターは5人とする
・1パーティは1〜3人までとする
・戦闘中の状態をもつ
  毒、麻痺、沈黙、混乱、石化
・戦闘中のステータス補正をもつ
バフ、デバフ
・戦闘中にアイテムを使える
・固有のスキルわざを持つ

class character {
  std::string character_name
  struct status {
    int32_t Lv;
    int32_t HP;
    int32_t MP;
    int32_t SP;
    int32_t ATK;
    int32_t DEF;
    int32_t AGI;
    int32_t LUK;
  }
}
